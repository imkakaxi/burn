colorForKey

NSColor* colorForKey(NSString* theKey);

Returns the colour for a given user defaults key or nil if the key does not exist or if it does not reference a colour.


Inputs

theKey
The key for the colour to be read from user defaults.

convertUserDefaults

void convertUserDefaults();

Converts the user defaults to the new format. The function is called from AppController's +initialize method.
Attention: This function uses the developer's knowledge about how the user defaults were structured before and how they are now. This knowledge is not intended for the rest of the application, but every part should treat its own part of the defaults only and should not care about the rest!

which

NSString* which(NSString* name);

This function returns the full path to a file. It is basically the same as the 'which' command in a Un*x shell. which searches the standard search path.

Inputs

name
The file/program that is to be searched.
