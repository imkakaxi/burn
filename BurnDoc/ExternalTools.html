Inputs

The index of the track to be ripped. Is 1-based.

The CD reader device from which the data is to be read. The ripper can rely on the correct CD to be in this device.
outFile
The name of the file for the output. The file format must be.wav For Burn to able to use it. The file name is a must, otherwise Burn.app will never find the file again. An external ripper which cannot rip single tracks in a fashion like this can therefore not be used.

mediaInformation: 

- (NSDictionary*) mediaInformation: (NSDictionary*)parameters;

Returns information about an inserted media.

The dictionary must contain at least an entry for the following key:

  • "type" _("Unknown"), "CD-R", "CD-RW" or _("NONE")
If "type" is "CD-R" or "CD-RW" there must be at least entrys for the following keys:
  • "capacity" a string value containing the capacity
  • "vendor" a string value containing vandor/manuf. info
  • "speed" a string value containing the writing speed
  • "empty" must be "yes" or "no"
  • "remCapacity" a string value containing the remaining capacity for multi-session media
  • "sessions" a string value containing the number of sessions
  • "appendable" tells whether sessions can be appended, must be "yes" or "no"

Inputs
parameters
The parameter list to be used when executing the backend program.

ExternalTools types

EBlankingMode

typedef enum ... EBlankingMode;

The currently supported modes for blanking a CD-RW.

  • fullBlank: The whole disk is blanked.
  • fastBlank: Only PMA and TOC are blanked.

EProcessStatus

typedef enum ... EProcessStatus;

Defines the stages a ripping/burning process may be in.

  • isStopped: The process is idle. No errors.
  • isRipping: The process is ripping one or more files.
  • isWaiting: The process is waiting for the actual process to begin (e.g. preparing the burning).
  • isPreparing: The process is writing the lead-in.
  • isBurning: The process is burning one or more files to the CD.
  • isFixating: The process is fixating (closing) the CD -> Lead-out.
  • isCancelled: The process has been cancelled by user action or by error.

SBurnerStatus

typedef struct ... SBurnerStatus;

Contains the current status of the burning process.

  • processStatus: The stage of the process as defined in EProcessStatus.
  • trackNumber: The number of the track currently processed. 1-based
  • trackProgress: The progress of the current track in percent (0 to 100).
  • entireProgress: The progress of the entire process in percent (0 to 100).
  • bufferLevel: The fill level of the burn buffer in percent (0 to 100).
